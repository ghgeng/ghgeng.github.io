Java JTextField

# JTextField 文本框组件

JTextField :单行文本框 ;类似html <input> 标签
JTextField类的对象是一个文本组件，它允许编辑单行文本。它继承了JTextComponent类。

# 常用构造函数:

Constructor Description
JTextField() 创建一个新的TextField
JTextField(String text) 创建一个用指定文本初始化的新TextField
JTextField(String text, int columns) 创建一个用指定文本和列初始化的新TextField
JTextField(int columns) 创建具有指定列数的新空TextField。

# 常用方法

Methods Description
void addActionListener(ActionListener l) 它用于添加指定的操作侦听器，以接收来自此文本字段的操作事件。
Action getAction() 它返回当前为这个ActionEvent源设置的Action，如果没有设置Action则返回null。
void setFont(Font f) 用于设置当前字体。
void removeActionListener(ActionListener l) 它用于删除指定的操作侦听器，以便它不再从此文本字段接收操作事件。

# 示例代码1：

  /**
     * JTextField :单行文本框 ;类似html <input> 标签
     */
    public void _05() {
        JTextField jTextField = new JTextField(20);
        root.add(jTextField);

        JButton jButton = new JButton("提交");
        root.add(jButton);
        // 添加事件
        jButton.addActionListener( (e) ->{
            // 获取输入框内容
            String text = jTextField.getText();
            System.out.println(text);
        });

        // 显示窗口
        this.setVisible(true);
    }

# 示例代码2：

package code.swing.learn;

import javax.swing.*;

/**
 * @author :ghgeng
 * @date 2023/7/3 22:55
 */
public class JTextFieldText extends JFrame {

    public JTextFieldText() {
        JTextField jTextField = new JTextField("我是第一个数据");
        jTextField.setBounds(50, 50, 200, 30);
        this.add(jTextField);

        JTextField jTextField2 = new JTextField("我是第二个数据");
        jTextField2.setBounds(50, 100, 200, 30);
        this.add(jTextField2);

        this.setTitle("JTextFieldText");
        this.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        this.setLocationRelativeTo(null);
        this.setSize(400, 400);
        this.setLayout(null);
        this.setVisible(true);
    }

    public static void main(String[] args) {
        new JTextFieldText();
    }
}
