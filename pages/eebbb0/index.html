Java JTextArea

# JTextArea

JTextArea类的对象是显示文本的多行区域。它允许编辑多行文本。它继承了JTextComponent类

# 常用构造函数：

Constructor Description
JTextArea() 创建初始不显示文本的文本区域。
JTextArea(String s) 创建初始显示指定文本的文本区域。
JTextArea(int row, int column) 创建具有指定行数和列数的文本区域，该区域最初不显示文本。
JTextArea(String s, int row, int column) 创建具有显示指定文本的指定行数和列数的文本区域。

# 常用方法:

Methods Description
void setRows(int rows) 用于设置指定的行数。
void setColumns(int cols) 用于设置指定的列数。
void setFont(Font f) 用于设置指定的字体。
void insert(String s, int position) 它用于在指定位置插入指定的文本。
void append(String s) 它用于将给定的文本追加到文档的末尾。

# 示例代码2：

package code.swing.learn;

import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

/**
 * @author :ghgeng
 * @date 2023/7/6 21:56
 */
public class JTextAreaTest {

    public static void main(String[] args) {
        JFrame jFrame = new JFrame("JTextAreaTest");

        JLabel jLabel1 = new JLabel();
        jLabel1.setBounds(20, 20, 100, 30);

        JLabel jLabel2 = new JLabel();
        jLabel2.setBounds(20, 60, 100, 30);

        JButton jButton = new JButton("划词");
        jButton.setBounds(20, 300, 100, 30);


        JTextArea jTextArea = new JTextArea("Hello World adc");
        jTextArea.setBounds(20, 100, 200, 200);

        jButton.addActionListener(e -> {
            String text = jTextArea.getText();
            String[] split = text.split("\\s");
            jLabel1.setText("单词数量: " + split.length);
            jLabel2.setText("长度为:" + text.length());
        });

        jFrame.add(jLabel1);
        jFrame.add(jLabel2);
        jFrame.add(jButton);
        jFrame.add(jTextArea);

        jFrame.setLayout(null);
        jFrame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        jFrame.setLocationRelativeTo(null);
        jFrame.setSize(400, 400);
        jFrame.setVisible(true);
    }
}
