git安装

# 1.git安装包安装

可能会出现git找不到命令情况

git安装 (opens new window)

  • 官网下载centos安装包 (opens new window)

# 2.安装包安装

需要配置各种环境变量，能连接外网推荐用yum安装

# 2.1安装前首先得安装依赖环境

yum install -y perl-devel
1

如果没有安装的话，会报如下错误：

make[1]: *** [perl.mak] Error 2
make: *** [perl/perl.mak] Error 2
1
2

# 2.2.下载源码包到 CentOS 服务器后进行解压

tar -zxvf git-2.9.5.tar.gz
cd git-2.9.5
1
2

安装目录

# 2.3.执行如下命令进行编译安装

# 检查环境，配置安装路径
./configure --prefix=/usr/local/git
1
2
# 编译安装
make && make install
1
2

# 2.4.添加到系统环境变量

打开环境变量

vim ~/.bashrc
1

文件末尾添加如下内容,wq保存：

# git
export PATH="/usr/local/git/bin:$PATH"
1
2

# 2.5.刷新变量，使配置生效

source ~/.bashrc
1

# 2.6.查看效果

git version
git version 2.45.1
1
2

# 3.yum安装

yum -y install git
1

  • 如果想卸载Git，需要运行sudo yum -y remove git
 yum -y remove git
1
