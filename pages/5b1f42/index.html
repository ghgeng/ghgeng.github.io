# 7、假如将当前容器实例删除，再重新来一次，之前建的dbo1实例还有吗?

# 7.1 上传到阿里云

  • 进入到自己仓库

  • 如果需要把数据库数据一并上传到阿里云，需要修改my.cnf默认的datadir目录，改为别的文件夹，不要使用默认文件夹

  • mysql 默认启动文件夹目录log:/var/log/mysql 、数据文件：/var/lib/mysql、my.cnf文件:/etc/mysql

  • 修改完之后，把主机的文件复制到docker容器内 docker cp /usr/local/mysql/data/ 容器ID:/data/mysql

  • 重启docker restart容器ID

    • 打包上传阿里云
    • 还是走阿里云仓库命令，重新docker tar 或者docker commit ，然后push到阿里云
    • docker commit -m="提交信息" -a="组综合" 容器id/镜像id 目标镜像名:[标签名]
    • docker tag [ImageId] registry.cn-hangzhou.aliyuncs.com/ghgeng/mysql_8.0:[镜像版本号]
    • docker push registry.cn-hangzhou.aliyuncs.com/ghgeng/mysql_8.0:[镜像版本号]
