模板语法

提示

指令由 v- 作为前缀，表明它们是一些由 Vue 提供的特殊 attribute

# 模板的理解

html 中包含了一些 JS 语法代码，语法分为两种，分别为：

    1. 插值语法（双大括号表达式）
    1. 指令（以 v-开头）
# 1、写一个<a>连接标签，想在href中的url获取到vue实例的url，目前是获取不到的，vue很早版本之前是可以用差值获取

<a href={ {url}}>点我 </a>

<a href="url">点我</a>
1
new Vue({
    el: '#root',
    data: {
        name: 'jack',
        url: 'http://www.baidu.com'
    }
})

# 2、需要再href前面加一个v-bind 或者用冒号:简写的形式绑定，不是所有的都简写成冒号，vue就会把引号中的作为表达式去解析，这是单项绑定。
 <a v-bind:href="url">点我 </a>
或者
 <a :href="url">点我 </a>

# 总结

  • 什么时候用差值？什么时候用指令?
1.插值语法：
    功能：用于解析标签体内容。
    写法：{{xxx}}，xxx是js表达式，且可以直接读取到data中的所有属性。
2.指令语法：
    功能：用于解析标签（包括：标签属性、标签体内容、绑定事件.....）。
    举例：v-bind:href="xxx" 或  简写为 :href="xxx"，xxx同样要写js表达式，
             且可以直接读取到data中的所有属性。
    备注：Vue中有很多的指令，且形式都是：v-????，此处我们只是拿v-bind举个例子。
