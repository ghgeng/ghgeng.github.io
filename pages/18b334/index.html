# Launch4j

Launch4j 需要和jre一起使用，没有安装的jre环境的需要安装一下
使用Launch4j将Java程序打包成可执行的.exe文件的过程如下：

# 1.下载和安装Launch4j：

  • 你可以从Launch4j的官方网站http://launch4j.sourceforge.net/ (opens new window)下载最新版本的Launch4j。

  • 下载适合你操作系统的版本，并按照说明进行安装。

  • 下载文件并解压，同时把jre复制到同目录下

# 2.准备Java可执行JAR文件：

确保你已经编写了Java程序，并将其打包成了可执行的JAR文件。按步骤导出一个可执行的jar文件，有jar可以跳过

  • 2.1这里准备了一个在线2048游戏 (opens new window)

  • 2.2导入文件，新建一个普通java项目

  • 2.3点击下一步

  • 2.4新建文件夹

  • 2.5 设置java资源文件，否则启动找不到图片和样式

  • 2.6 设置resource为根资源文件，然后点击保存

  • 2.7 项目结构

  • 2.8 启动程序，报错的话查看路径地址

先能启动起来，然后开始打jar包

  • 2.9 打jar包 在线文章 (opens new window)

# 3.打开Launch4j：

双击Launch4j的可执行文件以打开Launch4j界面，后边有问题以管理员运行。

# 4.配置Launch4j：

  • 4.1页面介绍

  • 4.2先写配置，简单的示例

在主界面上，选择“配置文件”选项卡或点击“配置”按钮进入配置界面。
在“基本信息”部分中，输入应用程序的名称和版本号。
在“Java设置”部分中，选择Java运行时环境的路径（即JRE或JDK的路径）。
在“类路径”部分中，添加你的JAR文件以及所有依赖项的路径。
如果需要，你可以在“JVM参数”部分中添加JVM参数。
在“界面”部分中，选择应用程序的图标。
在“输出文件”部分中，指定输出文件的路径和文件名，这将是你生成的.exe文件的路径和名称。

  • 4.3把jre复制到同级目录下

# 5.生成.exe文件：

图标转换 (opens new window)

点击“打包”按钮或类似的按钮（在某些版本中可能是“创建EXE”按钮），等待打包完成。

  • 5.1点击小齿轮导出,填写导出配置内容，下次可以直接打开.xml文件，不需要再配置

# 6.运行生成的.exe文件

  • 在你指定的输出文件夹中找到生成的.exe文件，并双击运行它。
  • 注意：在运行Launch4j之前，确保你的系统上已经安装了Java运行时环境（JRE），因为Launch4j需要Java来运行你的JAR文件。此外，如果你想要通过配置参数在Java程序的main(String[] args)方法中获取到参数，你可以在Launch4j的“命令行参数”部分进行设置，多个参数之间使用空格分隔。
