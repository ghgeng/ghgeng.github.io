JavaFX BorderPane

# BorderPane

提示

BorderPane 是 JavaFX 中的一个布局容器，它将布局空间分为五个区域：顶部（top）、左侧（left）、底部（bottom）、右侧（right）和中心（center）。这种布局非常适合那些需要明确区分头部、侧边栏、底部栏和主要内容区域的用户界面。

在 BorderPane 中，你可以将不同的节点（如按钮、文本框、面板等）添加到这些区域中。通常，中心区域会占据大部分空间，而顶部、底部、左侧和右侧区域则较小，用于放置导航栏、工具栏、状态栏等。

# BorderPane 的特点

  • 提供了清晰的区域划分，使得布局更加直观和易于管理。
  • 中心区域通常占据剩余空间，而其他四个区域则保持其固定大小或首选大小。
  • 支持动态调整大小，当窗口大小改变时，中心区域会相应地扩展或收缩，而边缘区域则保持其位置或大小（取决于具体设置）。

# 示例代码

package code.javafx.learn.JavaFX布局;

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.BorderPane;
import javafx.stage.Stage;

public class BorderPaneExample extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 创建一个BorderPane实例
        BorderPane borderPane = new BorderPane();

        // 创建要添加到各个区域的节点
        Button topButton = new Button("顶部");
        Button leftButton = new Button("左侧");
        Button bottomButton = new Button("底部");
        Button rightButton = new Button("右侧");
        Button centerButton = new Button("中心");

        // 将节点添加到BorderPane的相应区域
        borderPane.setTop(topButton);
        borderPane.setLeft(leftButton);
        borderPane.setBottom(bottomButton);
        borderPane.setRight(rightButton);
        borderPane.setCenter(centerButton);


        // 设置场景，并将BorderPane添加到场景中
        Scene scene = new Scene(borderPane, 400, 300);

        // 配置并显示舞台
        primaryStage.setTitle("BorderPane 示例");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
