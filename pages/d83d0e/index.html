JavaFX DialogPane

# DialogPane

提示

在JavaFX中，DialogPane 是 Dialog 类的一个内部组件，用于封装对话框的内容，包括标题、内容文本、图形节点、按钮等。虽然 DialogPane 本身不是公开类（即你不能直接实例化它），但你可以通过 Dialog 类的API来访问和配置它的不同部分。

# 示例代码

package code.javafx.learn.JavaFX布局;

import javafx.application.Application;
import javafx.geometry.Insets;
import javafx.scene.Node;
import javafx.scene.control.Alert;
import javafx.scene.control.Button;
import javafx.scene.control.ButtonBar;
import javafx.scene.control.ButtonType;
import javafx.scene.control.Dialog;
import javafx.scene.control.Label;
import javafx.scene.control.TextField;
import javafx.scene.layout.StackPane;
import javafx.scene.layout.VBox;
import javafx.stage.Stage;

public class DialogPaneExample extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 创建一个Dialog
        Dialog<Void> dialog = new Dialog<>();
        dialog.setTitle("Confirmation Dialog");
        dialog.setHeaderText("Are you sure?");

        // 访问DialogPane并添加自定义内容
        // 注意：我们直接通过Dialog的API来设置内容，而不是直接访问DialogPane
        VBox content = new VBox(10);
        content.setPadding(new Insets(10));

        // 添加一些自定义内容
        Label label = new Label("This is a custom message:");
        TextField textField = new TextField();
        textField.setPromptText("Enter your reason here...");

        content.getChildren().addAll(label, textField);

        // 设置Dialog的内容
        dialog.getDialogPane().setContent(content);

        // 配置按钮
        ButtonType buttonTypeOk = new ButtonType("OK", ButtonBar.ButtonData.OK_DONE);
        ButtonType buttonTypeCancel = new ButtonType("Cancel", ButtonBar.ButtonData.CANCEL_CLOSE);

        dialog.getDialogPane().getButtonTypes().addAll(buttonTypeOk, buttonTypeCancel);

        // 处理按钮点击事件
        dialog.setResultConverter(dialogButton -> {
            if (dialogButton == buttonTypeOk) {
                // 用户点击了OK，可以在这里处理相关逻辑
                System.out.println("User entered: " + textField.getText());
                return null; // 或者返回一些有用的结果
            }
            return null; // 用户点击了Cancel或关闭了对话框
        });

        // 显示对话框
        dialog.showAndWait();

        // 注意：这里不需要设置场景和舞台给Dialog，因为Dialog是模态的，并且会管理自己的显示

        // 示例结束，通常你会在这里继续设置primaryStage的内容并显示它
        // 但为了这个示例的简洁性，我们只显示了Dialog
    }

    public static void main(String[] args) {
        launch(args);
    }
}

  • 在这个示例中，我们创建了一个 Dialog 对象，并设置了它的标题和头部文本。
  • 然后，我们构建了一个自定义的 VBox 布局，其中包含了一个 Label 和一个 TextField，并将这个 VBox 设置为了对话框的内容。
  • 接下来，我们添加了两个按钮类型到对话框的按钮栏中，并配置了当点击这些按钮时应该执行的操作。最后，我们通过调用 showAndWait() 方法来显示对话框，并等待用户做出响应。
