Java JComponent

# JComponent

JComponent类是除了顶层容器之外的所有Swing组件的基类。名称以“J”开头的Swing组件是JComponent类的后代。例如，JButton, JScrollPane, JPanel, JTable等。但是，JFrame和JDialog不继承JComponent类，因为它们是顶层容器的子类。

JComponent类扩展了Container类，而Container类又扩展了Component。Container类支持向容器中添加组件。

# 常用方法:

Method Description
setActionMap(ActionMap am) 它将ActionMap设置为am。
setBackground(Color bg) 它设置了该组件的背景颜色。
setFont(Font font) 它为这个组件设置字体。
setMaximumSize(Dimension maximumSize) 它将此组件的最大大小设置为一个常量值。
setMinimumSize(Dimension minimumSize) 它将此组件的最小大小设置为一个常数值。
setUI(ComponentUI newUI) 它为这个组件设置外观委托。
setVisible(boolean aFlag) 它使组件可见或不可见。
setForeground(Color fg) 它设置该组件的前景色。
getToolTipText(MouseEvent event) 它返回要用作事件工具提示的字符串。
getTopLevelAncestor() 它返回该组件(包含窗口或Applet)的顶级祖先，如果该组件未被添加到任何容器中，则返回null。
getTransferHandler() 它获得transferHandler属性。

# 示例代码1：

package code.swing.learn;

import javax.swing.*;
import java.awt.*;

/**
 * @author :ghgeng
 * @date 2023/7/20 16:17
 */
public class JComponentTest extends JComponent {

    @Override
    public void paint(Graphics g) {
        g.setColor(Color.red);
        g.fillRect(30, 30, 100, 30);
    }

    public static void main(String[] args) {
        JComponentTest jComponentTest = new JComponentTest();

        JFrame jFrame = new JFrame("JComponent Example");

        jFrame.setSize(400, 400);
        jFrame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        jFrame.add(jComponentTest);
        jFrame.setLocationRelativeTo(null);
        jFrame.setVisible(true);

    }
}
