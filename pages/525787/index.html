JavaFX TilePane

# TilePane

提示

在JavaFX中，TilePane是一个布局容器，它按照网格方式排列其子节点，但允许子节点具有不同的大小。TilePane非常适合于需要展示大小可能不统一的项（如图片、按钮等）的场景。以下是一个TilePane的代码示例，展示了如何使用它以及如何在其中添加和排列子节点。

# 示例代码


package code.javafx.learn.JavaFX布局;

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.TilePane;
import javafx.stage.Stage;

public class TilePaneExample extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 创建一个TilePane
        // 你可以通过构造函数指定行数和列数的提示（注意：这只是提示，实际布局会根据子节点的大小和TilePane的大小动态调整）
        TilePane tilePane = new TilePane(10, 10); // 这里的10, 10是水平和垂直间距，不指定行列数

        // 创建一些子节点（这里以按钮为例）
        Button btn1 = new Button("Button 1");
        btn1.setMinSize(100, 50); // 设置按钮的最小尺寸，以便观察效果

        Button btn2 = new Button("Button 2");
        btn2.setMinSize(150, 100); // 另一个按钮具有不同的尺寸

        Button btn3 = new Button("Button 3");
        btn3.setMinSize(120, 80); // 第三个按钮也有其独特的尺寸

        // 将子节点添加到TilePane中
        tilePane.getChildren().addAll(btn1, btn2, btn3);

        // 设置场景和舞台
        Scene scene = new Scene(tilePane, 300, 200); // 场景大小为300x200

        primaryStage.setTitle("TilePane Example"); // 舞台标题
        primaryStage.setScene(scene); // 将场景设置给舞台
        primaryStage.show(); // 显示舞台
    }

    public static void main(String[] args) {
        launch(args); // 启动JavaFX应用程序
    }
}

  • 在这个示例中，我们创建了一个TilePane，但没有在构造函数中指定行数和列数，因为TilePane会根据子节点的大小和自身的可用空间来动态调整布局。我们给TilePane设置了水平和垂直间距（在这个例子中是10像素），这有助于在子节点之间留出空间。
  • 然后，我们创建了三个Button作为子节点，并为它们设置了不同的最小尺寸。这些尺寸会影响它们在TilePane中的布局方式。
  • 最后，我们将这些子节点添加到TilePane中，设置了场景和舞台的大小，并显示了舞台。
