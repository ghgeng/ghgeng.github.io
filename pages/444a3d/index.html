JavaFX Color

提示

在JavaFX中，我们可以用颜色填充形状。我们可以灵活地使用几个方法创建自己的颜色，并将其作为Paint对象传递给setFill()方法。让我们讨论一下在JavaFX中创建颜色的几种方法。

# RGB Color

  • RGB颜色系统是在图形中创建颜色的最流行的方法。它由三个组件组成，分别为RED→R, GREEN→G和BLUE→b。每个组件使用8位，这意味着每个组件可以具有从0到22^8 - 1=255的整数值。
  • 计算机屏幕可以看作是像素的集合。这个集合(R,G,B)实际上代表了它们各自的led在屏幕上的发射。
  • 如果红色的值被设置为0，那么这意味着红色LED被关闭，而值255表示LED的全部发射在那里。(0,0,0)的组合表示
  • 使用RGB的叠加，我们可以表示255255255种不同的颜色。在JavaFX中，JavaFX .scene.paint. color类表示颜色。
  • Color类有一个名为rgb()的静态方法。它接受三个整数参数Red, Green, Blue和一个可选的双精度参数alpha。alpha值与颜色的不透明度成正比。alpha值0表示颜色是完全透明的，而值1表示颜色是完全不透明的。

# 示例

package code.javafx;

import javafx.application.Application;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.effect.DropShadow;
import javafx.scene.paint.Color;
import javafx.scene.shape.QuadCurve;
import javafx.scene.shape.Rectangle;
import javafx.stage.Stage;

/**
 * @author :ghgeng
 * @date 2023/9/20 17:39
 */
public class JavaFXColor extends Application {

    @Override
    public void start(Stage primaryStage) throws Exception {
        primaryStage.setTitle("JavaFXColor");
        Group group = new Group();

        Rectangle rect = new Rectangle();
        rect.setX(50);
        rect.setY(20);
        rect.setWidth(100);
        rect.setHeight(150);
        int red=20;
        int green=125;
        int blue=10;
        rect.setFill(Color.rgb(red, green, blue,0.3));

        group.getChildren().addAll(rect);
        Scene scene = new Scene(group, 300, 400, Color.ANTIQUEWHITE);
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
