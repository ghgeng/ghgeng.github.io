              tail命令

              # 常用参数

              • 命令模式

              tail [OPTION]... [FILE]...

              命令 功能描述
              -c 输出文件尾部的N（N为整数）个字节内容
              -f 持续显示文件最新追加的内容
              -n <N> 输出文件的尾部N（N位数字）行内容

              # 常用示例

              # 当文件增长时，动态查看文件追加的内容

              tail -f a.txt
              1

              # 输出文件最后10行

              tail /etc/passwd
              1
