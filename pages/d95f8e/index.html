JavaFX TabPane

# JavaFX TabPane

提示

TabPane 是 JavaFX 中的一个非常有用的组件，它允许开发者在单个窗口中创建多个标签页（Tab），每个标签页都可以包含不同的内容，如文本、图像、按钮或其他复杂的布局。这使得 TabPane 成为构建具有多个视图或工作区的桌面应用程序时的理想选择。

  • TabPane：这是主要的容器，负责管理和显示所有的标签页。
  • Tab：每个 Tab 是 TabPane 中的一个子项，它代表了一个单独的视图或页面。每个 Tab 可以有自己的标题（text）和图标（graphic），并且包含了一个节点（Node），这个节点通常是某种布局（如 VBox, HBox, Pane 等），其中可以包含更多的UI组件。

# 示例代码

package code.test;

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Tab;
import javafx.scene.control.TabPane;
import javafx.scene.control.TextArea;
import javafx.scene.layout.VBox;
import javafx.stage.Stage;

public class TabPaneExample extends Application {

    @Override
    public void start(Stage primaryStage) {
        // 创建TabPane
        TabPane tabPane = new TabPane();

        // 创建第一个Tab，包含一个TextArea
        Tab tab1 = new Tab("Tab 1");
        TextArea textArea1 = new TextArea();
        textArea1.setText("这是第一个标签页的内容。");
        tab1.setContent(textArea1);

        // 创建第二个Tab，包含一个VBox，VBox里包含两个TextArea
        Tab tab2 = new Tab("Tab 2");
        VBox vBox = new VBox(10); // 10是子项之间的间距
        TextArea textArea2 = new TextArea();
        textArea2.setText("这是第二个标签页的第一个TextArea。");
        TextArea textArea3 = new TextArea();
        textArea3.setText("这是第二个标签页的第二个TextArea。");
        vBox.getChildren().addAll(textArea2, textArea3);
        tab2.setContent(vBox);

        // 将Tab添加到TabPane
        tabPane.getTabs().addAll(tab1, tab2);

        // 设置场景和舞台
        Scene scene = new Scene(tabPane, 400, 300);
        primaryStage.setTitle("TabPane 示例");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

  • 在这个例子中，SplitPane被设置为水平分割（HORIZONTAL_SPLIT），这意味着你可以通过拖动分隔符来调整左右两边组件的宽度。setDividerPositions(0.3)用于设置分隔符的初始位置，使得左侧的TextArea占据整个SplitPane宽度的30%。你可以通过调整这个值来改变初始布局。

  • 这个简单的示例展示了如何使用SplitPane来创建可调整大小的界面元素，这对于创建用户友好的GUI应用程序来说是非常有用的。
