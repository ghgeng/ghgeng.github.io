# 2.如果版本太低更新node版本

你需要添加NodeSource的库到你的包管理器。你可以根据你需要的Node.js版本来选择添加。例如，如果你想要安装Node.js 18.x，你可以使用以下命令
这里18就是版本，可修改指定版本

curl -sL https://rpm.nodesource.com/setup_18.x | sudo bash -
1
