工厂+策略

# 业务场景

支付方式有支付宝支付、微信支付、网银支付等等，如果在代码中通过不断的写if-elseif去实现的话，虽然看起来更直观，但实际上不利于后续的扩展和维护，后期如果由于业务需要使用混合支付的话，就需要改动if-elseif的逻辑，违反了开闭原则，代码如下:

String payTypeCode = "WX";
if ("WX".equals(payTypeCode)){
    System.out.println("微信支付");
}else if ("ALI".equals(payTypeCode)){
    System.out.println("支付宝支付");
}else if ("WANGYIN".equals(payTypeCode)){
    System.out.println("网银支付");
}else {
    System.out.println("混合支付支付");
}

# 解决方案

  • 通过工厂模式+策略模式+自定义注解的方式彻底消除if-elseif，提高代码的可扩展性，最终达到实现开闭原则的目的

# 新建支付方式的枚举

项目结构


@Getter
@AllArgsConstructor
public enum PayEnum {

    WX("WX", "微信支付"),

    ZFB("ZFB", "支付宝支付");

    private String type;

    private String msg;


    public static PayEnum convertPayEnum(String type) {
        for (PayEnum payEnum : values()) {
            if (payEnum.getType().equals(type)) {
                return payEnum;
            }
        }
        return null;
    }
}

# 新建支付服务策略工厂

@Component
public class PayServiceFactory implements ApplicationContextAware {

    //支付方式枚举-支付服务的映射集合
    private static Map<PayEnum, PayService> map = new HashedMap();

    // 初始化支付枚举-支付服务的映射
    @Override
    public void setApplicationContext(ApplicationContext applicationContext) throws BeansException {

        Map<String, Object> beansWithAnnotation = applicationContext.getBeansWithAnnotation(PayAnnotation.class);
        if (ObjectUtil.isEmpty(beansWithAnnotation)) {
            throw new RuntimeException("支付服务映射初始化失败");
        }

        beansWithAnnotation.forEach((k, v) -> {
            if (!(v instanceof PayService)) {
                throw new RuntimeException("注解" + PayAnnotation.class + "只能用于" + PayService.class + "实现类中!");
            }
            PayService payService = (PayService) v;
            PayAnnotation annotation = payService.getClass().getAnnotation(PayAnnotation.class);
            map.put(annotation.value(), payService);
        });
    }

    // 工厂方法获取支付服务实现
    public static PayService getPayService(String patCode) {
        PayEnum payEnum = PayEnum.convertPayEnum(patCode);
        if (payEnum == null) {
            throw new RuntimeException("没有符合类型的支付方式");
        }
        PayService payService = map.get(payEnum);
        if (payService == null) {
            throw new RuntimeException("没有符合支付方式的实现类");
        }
        return payService;
    }
}

# 创建测试类

@Controller
public class DesignController {

    /**
     * 测试策略接口
     *
     * @param payType
     */
    @RequestMapping(value = "/payController", method = RequestMethod.GET)
    @ResponseBody
    public void PayController(@RequestParam("payType") String payType) {
        PayService payService = PayServiceFactory.getPayService(payType);
        payService.pay();
    }
}
