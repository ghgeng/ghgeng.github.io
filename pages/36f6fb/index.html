BlockingQueue详解

# BlockingQueue详解

提示

BlockingQueue 通常用于一个线程生产对象，而另外一个线程消费这些对象的场景。

  • 常用api方法
方法类型 抛异常 特殊值 阻塞 超时
插入 add(o) offer(o) put(o) offer(o, timeout, timeunit)
移除 remove() poll() take() poll(timeout, timeunit)
检查 element() peek()
  • 解释:

1.抛异常: 如果试图的操作无法立即执行，抛一个异常。

当阻塞队列满时，再往队列里add元素会抛出IllegalStateException：Queue full；
当阻塞队列空时，再从队列里remove移除元素会抛NoSuchElementException。

2.特定值: 如果试图的操作无法立即执行，返回一个特定的值(常常是 true / false)。

插入成功，返回true；插入失败，返回false；
删除成功返回出队列元素；删除失败返回null；

3.阻塞(添加无返回值): 如果试图的操作无法立即执行，该方法调用将会发生阻塞，直到能够执行。

当阻塞队列满时，生产者线程继续往队列里put元素，队列会一直阻塞生产线程直到put数据or响应中断退出。
当阻塞队列空时，消费者线程试图take队列里的元素，队列会一直阻塞消费者线程直到队列有可用元素。

4.超时: 如果试图的操作无法立即执行，该方法调用将会发生阻塞，直到能够执行，但等待时间不会超过给定值。 返回一个特定值以告知该操作是否成功(典型的是 true / false)。

当向阻塞队列offer元素时候，时间超过了设定的值，就会出现超时中断；
当向阻塞队列poll元素时候，时间超过了设定的值，就会出现超时中断。

# BlockingQueue例子

package code.test;

import lombok.Data;
import java.util.concurrent.ArrayBlockingQueue;
import java.util.concurrent.BlockingQueue;

/**
 * @author :ghgeng
 * @date 2022/3/5 10:51
 */
public class Test {

    private static Object object;

    public static void main(String[] args) throws Exception {

        BlockingQueue<A> as = new ArrayBlockingQueue<>(3);

        Producer po = new Producer(as);
        Consumer co = new Consumer(as);

        new Thread(po).start();
        new Thread(co).start();
        Thread.sleep(4000);
    }

    
    @Data
    static
    class A {
        private String name;
        private Integer age;
    }


    // 定义消费者
    @Data
    static
    class Consumer extends Thread{

        private BlockingQueue<A> queue = null;
        public Consumer(BlockingQueue queue){
            this.queue = queue;
        }

        @Override
        public void run() {
            try {
                System.out.println( queue.take());
                System.out.println( queue.take());
                System.out.println( queue.take());
            } catch (InterruptedException e) {
                e.printStackTrace();
            }

        }
    }
    

    // 定义消费者
    @Data
    static
    class Producer implements Runnable{

        private BlockingQueue<A> queue = null;

        public Producer(BlockingQueue queue){
            this.queue = queue;
        }
        @Override
        public void run() {
            try {
                A a = new A();
                a.setName("1");
                queue.put(a);
                Thread.sleep(100);
                a.setName("b");
                queue.put(a);
                Thread.sleep(100);
                a.setName("c");
                queue.put(a);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
    }

};
