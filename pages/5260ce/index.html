              # 3、添加自定义ll命令

              有时候我们在运行如 ll 命令的时候系统会提示如下内容。

              [root@23cfef8de20d /]# ll
bash: ll: command not found
              1
              2
              • 这是因为ll并不是系统中的基本命令，ll是ls -l的别名，那么这个我们只需要编辑一下/etc/skel/.bashrc文件添加命令就可以了，如下所示。

              1、vim /root/.bashrc
              2、添加 alias ll='ls -l' 到文件配置中
              3、source /root/.bashrc

              • 保存退出如下配置:
              # .bashrc
  
# User specific aliases and functions

alias rm='rm -i'
alias cp='cp -i'
alias mv='mv -i'
alias ll='ls -l'
# Source global definitions
if [ -f /etc/bashrc ]; then
        . /etc/bashrc
fi
