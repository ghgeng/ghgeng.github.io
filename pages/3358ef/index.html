Java JToggleButton

# JToggleButton

JToggleButton用于创建切换按钮，它是两种状态的按钮来打开或关闭。

# 内部类

Modifier and Type Class Description
protected class JToggleButton.AccessibleJToggleButton 这个类实现了JToggleButton类的可访问性支持。
static class JToggleButton.ToggleButtonModel ToggleButton模型。

# 常用构造函数：

Constructor Description
JToggleButton() 它在不设置文本或图像的情况下创建一个初始未选中的切换按钮。
JToggleButton(Action a) 它创建了一个切换按钮，其中的属性取自所提供的Action。
JToggleButton(Icon icon) 它创建一个初始未选中的切换按钮，带有指定的图像，但没有文本。
JToggleButton(Icon icon, boolean selected) 它创建一个带有指定图像和选择状态的切换按钮，但没有文本。
JToggleButton(String text) 它用指定的文本创建一个未选中的切换按钮。
JToggleButton(String text, boolean selected) 它创建一个具有指定文本和选择状态的切换按钮。
JToggleButton(String text, Icon icon) 它创建一个具有指定文本和图像的切换按钮，并且该按钮最初未被选中。
JToggleButton(String text, Icon icon, boolean selected) 它创建一个具有指定文本、图像和选择状态的切换按钮。

# 常用方法：

Method Description
getAccessibleContext() 它获取与这个JToggleButton关联的AccessibleContext。
getUIClassID() 它返回一个字符串，该字符串指定呈现该组件的l&f类的名称。
paramString() 它返回JToggleButton的字符串表示形式。
updateUI() 它将UI属性重置为当前外观的值。

# 示例代码1：

package code.swing.learn;

import javax.swing.*;
import java.awt.*;

/**
 * @author :ghgeng
 * @date 2023/7/18 23:43
 */
public class JToggleButtonTest {
    public static void main(String[] args) {
        JFrame jFrame = new JFrame("JTextAreaTest");

        JToggleButton jToggleButton = new JToggleButton("ON / OFF");
        jToggleButton.addItemListener(e -> {
            if (jToggleButton.isSelected()) {
                jToggleButton.setText("ON");
            } else {
                jToggleButton.setText("OFF");
            }
        });

        jFrame.add(jToggleButton);
        jFrame.setLayout(new FlowLayout());
        jFrame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        jFrame.setLocationRelativeTo(null);
        jFrame.setSize(400, 400);
        jFrame.setVisible(true);
    }
}
