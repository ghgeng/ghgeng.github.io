Java JSlider

# JSlider

Java JSlider类用于创建滑动条。通过使用JSlider，用户可以从特定范围中选择一个值。

# 常用构造函数：

Constructor Description
JSlider() 创建一个初始值为50，范围为0到100的滑块。
JSlider(int orientation) 创建一个由JSlider设置指定方向的滑动条。水平或JSlider。垂直，范围0到100，初始值50。
JSlider(int min, int max) 使用给定的最小值和最大值创建一个水平滑动条。
JSlider(int min, int max, int value) 使用给定的min, Max和value创建一个水平滑动条。
JSlider(int orientation, int min, int max, int value) 使用给定的方向，最小值，最大值和值创建一个滑动条。

# 常用方法：

Method Description
public void setMinorTickSpacing(int n) 用于设置滑块的小刻度间距。
public void setMajorTickSpacing(int n) 用于设置滑块的主要刻度间距。
public void setPaintTicks(boolean b) 用于确定是否绘制了刻度标记。
public void setPaintLabels(boolean b) 用于确定是否绘制标签。
public void setPaintTracks(boolean b) 用于确定是否绘制轨道。

# 示例代码1：

package code.swing.learn;

import javax.swing.*;
import java.awt.*;

/**
 * @author :ghgeng
 * @date 2023/7/17 17:42
 */
public class JSliderTest extends JFrame {

    JSliderTest() {

        JLabel jLabel = new JLabel("值为： ");
        jLabel.setBounds(30, 100, 200, 30);
        this.add(jLabel);

        JSlider jSlider = new JSlider(JSlider.HORIZONTAL, 0, 50, 25);
        // 添加监听事件
        jSlider.addChangeListener(e -> {
            int value = jSlider.getValue();
            jLabel.setText("值为： " + String.valueOf(value));
        });

        // 设置标尺大小
        jSlider.setMinorTickSpacing(5);
        // 设置刻度大小
        jSlider.setMajorTickSpacing(10);
        // 是否展示标尺
        jSlider.setPaintTicks(true);
        // 是否展示刻度数字
        jSlider.setPaintLabels(true);

        JPanel jPanel = new JPanel();
        jPanel.add(jSlider);

        this.add(jPanel);

        // 显示窗口
        this.setTitle("JSliderTest");
        this.setSize(400, 400);
        this.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        this.setLocationRelativeTo(null);
        this.setVisible(true);

    }

    public static void main(String[] args) {
        JSliderTest jSliderTest = new JSliderTest();
        // 使此窗口的大小自适应
//        jSliderTest.pack();
    }
}
