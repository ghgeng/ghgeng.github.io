              grep命令

              提示

              grep来自于英文词组“global search regular expression and print out the line”的缩写，意思是用于全面搜索的正则表达式，并将结果输出。人们通常会将grep命令与正则表达式搭配使用，参数作为搜索过程中的补充或对输出结果的筛选，命令模式十分灵活。

              # 常用参数

              • 命令模式

              grep [OPTION...] PATTERNS [FILE...]
              grep [OPTION...] -e PATTERNS ... [FILE...]
              grep [OPTION...] -f PATTERN_FILE ... [FILE...]

              命令 功能描述
              -c, --count 只输出匹配的行数，不是匹配字符串的个数
              --color, --colour[=WHEN] 将找到的关键词加上颜色显示。WHEN 可取值 never、always 或 auto
              -L, --files-without-match 只列出符合匹配的文件名，不列出具体的匹配行
              -o, --only-matching 只显示匹配的字符串，并以单独行输出
              -b, --byte-offset 在匹配的行之前，标示出该行第一个字符的位编号,即字符在文本中的字节偏移下标，包括换行符
              -n, --line-number 显示匹配行及行号
              -i, --ignore-case 忽略字符大小写
              -w, --word-regexp 匹配整词
              -x, --line-regexp 匹配整行
              -h 查询多文件时不显示文件名
              -l 只列出符合匹配的文件名，不列出具体的匹配行

              # 常用示例

              # 匹配全词

              grep -w "hello" fileName.txt
              1

              # 匹配单词的行并忽略大小写

              grep -iw "hello" fileName.txt
              1

              # 统计字段出现次数

              grep -o string aa.txt |wc -l
              1

              # 递归搜索当前目录下的所有文件

              grep -r "lvlv" .
              1

              # 搜索某个文件中，包含某个关键词的内容

              grep root /etc/passwd
              1

              # 搜索某个文件中，以某个关键词开头的内容

              grep ^root /etc/passwd
              1

              # 输出在某个文件中，包含某个关键词行的数量

              [root@linuxcool ~]# grep -c root /etc/passwd /etc/shadow

              # grep多管道命令查询

              grep "pattern1"  filename | grep "pattern2"
grep "key1" file.txt |grep "key2" |grep "key3"
              1
              2

              # grep过滤多个关键词并输出前10行内容

              grep "key1" file.txt |grep "key2" |grep "key3" |head -n 10
              1

              # grep过滤多个关键词并输出后10行内容

              grep "key1" file.txt |grep "key2" |grep "key3"|tail -n 10
              1

              # grep过滤多个关键词并输出10~20行内容

              # 在输出前面20行的内容前提基础下，再输出后面10行内容（20-10=10），就是输出10~20的内容了
grep "key1" file.txt |grep "key2" |grep "key3"|head -n 20 |tail -n 10
# 也可以用sed输出指定行
grep "key1" file.txt |grep "key2" |grep "key3"|sed -n '7,13p'
              1
              2
              3
              4

              # grep过滤多个关键词并输出10~20行内容到文件test.txt

              grep "key1" file.txt |grep "key2" |grep "key3"|head -n 20 |tail -n 10 >> test.txt
              1

              # 搜索多个文件并查找匹配文本在哪些文件中

              grep -l "text" [*] [file1 file2 file3...]
              1

              # 忽略匹配样式中的字符大小写

              echo "hello world" | grep -i "HELLO"
# hello
