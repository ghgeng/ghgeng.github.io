Java JSplitPane

# JSplitPane

JSplitPane用于划分两个组件。这两个组件是根据外观实现划分的，用户可以调整它们的大小。如果两个组件的最小大小大于拆分窗格的大小，则分隔器将不允许您调整其大小。
拆分窗格中的两个组件可以使用JSplitPane从左到右对齐。或者使用JSplitPane.VERTICAL_SPLIT从上到下。当用户调整组件的大小时，使用组件的最小尺寸来确定组件可以设置的最大/最小位置。

# 常用构造函数：

Constructor Description
JSplitPane() 它创建了一个新的JsplitPane，配置为横向排列并排的子组件，使用两个按钮来排列组件。
JSplitPane(int newOrientation) 它创建了一个新的JsplitPane，配置了指定的方向。
JSplitPane(int newOrientation, boolean newContinuousLayout) 它创建一个具有指定方向和重绘样式的新JsplitPane。
JSplitPane(int newOrientation, boolean newContinuousLayout, Component newLeftComponent, Component newRightComponent) 它创建一个新的JsplitPane，具有指定的方向和重绘样式，以及指定的组件。
JSplitPane(int newOrientation, Component newLeftComponent, Component newRightComponent) 它用指定的方向和指定的组件创建一个新的JsplitPane。

# 常用方法:

Method Description
addImpl(Component comp, Object constraints, int index) 它将指定的组件添加到此拆分窗格。
getAccessibleContext() 它获取与这个JSplitPane关联的AccessibleContext。
getDividerLocation() 它返回传递给setDividerLocation的最后一个值。
getDividerSize() 它返回除数的大小。
getBottomComponent() 它返回分隔符下方或右侧的组件。
getRightComponent() 它将组件返回到分隔符的右侧(或下方)。
isContinuousLayout() 它获得continuousLayout属性。
isOneTouchExpandable() 它获得oneTouchExpandable属性。
setOrientation(int orientation) 设置方向，或分离器的分割方式。

# 示例代码1：

package code.swing.learn;

import javax.swing.*;
import java.awt.*;

/**
 * @author :ghgeng
 * @date 2023/7/21 14:59
 */
public class JSplitPaneTest {

    public static void main(String[] args) {
        JFrame j = new JFrame("JScrollBarTest");
        // 创建两个下拉组件
        JComboBox jComboBox1 = new JComboBox(new String[]{"item1", "item2", "item3", "item4"});
        JComboBox jComboBox2 = new JComboBox(new String[]{"A", "B", "C", "D"});
        // 创建两个面板
        JPanel jPanel1 = new JPanel();
        jPanel1.add(jComboBox1);
        JPanel jPanel2 = new JPanel();
        jPanel2.add(jComboBox2);
        // 创建分割面板
        JSplitPane jSplitPane = new JSplitPane(JSplitPane.HORIZONTAL_SPLIT, jPanel1, jPanel2);

        j.add(jSplitPane);
        // 显示窗口
        j.setSize(400, 400);
        j.setLayout(new FlowLayout());
        j.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        j.setLocationRelativeTo(null);
        j.setVisible(true);
    }
}
