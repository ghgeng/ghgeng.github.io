# 当有用户选择时

当删除或者修改文件时，如下个修改高级权限

  • 文件右键 -> 属性 -> 安全
  • 选择要修改的用户，然后编辑修改

# 当有用户选择时

  • 出现 所有组或用户均不具有访问此对象的权限。但是该对象的所有者可以分配权限。
  • 点击编辑、新增用户 Administrators
  • 选择检查名称、保存即可
