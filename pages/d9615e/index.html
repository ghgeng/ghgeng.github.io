Java JDialog

# JDialog

JDialog控件表示一个带有边框和标题的顶层窗口，用于接收来自用户的某种形式的输入。它继承Dialog类。 与JFrame不同，它没有最大化和最小化按钮。

# 常用构造函数：

Constructor Description
JDialog() 它用于创建没有标题和没有指定Frame所有者的非模态对话框。
JDialog(Frame owner) 它用于创建一个非模态对话框，并将指定的Frame作为其所有者和空标题。
JDialog(Frame owner, String title, boolean modal) 它用于创建具有指定标题、所有者框架和模态的对话框。

# 示例代码：

package code.swing.learn;

import javax.swing.*;
import java.awt.*;

/**
 * @author :ghgeng
 * @date 2023/7/18 11:12
 */
public class JDialogTest {

    JDialogTest() {
        JFrame jFrame = new JFrame();

        JDialog jDialog = new JDialog(jFrame, "Dialog Test", true);

        jDialog.setLayout(new FlowLayout());

        JButton jButton = new JButton("ok");
        jDialog.add(jButton);
        jButton.addActionListener(e -> {
            jDialog.setVisible(false);
        });

        jDialog.add(jButton);
        jDialog.setSize(400, 400);
        jDialog.setVisible(true);
        jDialog.setLocationRelativeTo(null);
    }

    public static void main(String[] args) {
        new JDialogTest();
    }
}
