# 泛型：super、extend、？

  • super 是指定泛型的『下界』

    <? super T> 表示 任何是类型 T 的超类的类型。就是父类

  • extend 是指定泛型的『上界』

    <? extends T> 表示 任何继承自类型 T 的类型。就是子类

  • ？是通配符

    ? 是通配符，wildcard，这样在报错时就知道说的是 ? 通配符

# 示例 super

package code.Java基础.泛型;

public class GenericType {
    public static void main(String[] args) {
        AA<Other2> stringAA = new AA<>();
        // 只能是Other3类型、或者是Other3的父类
        fun(stringAA);
    }
    public static void fun(AA<? super Other3> temp){
        System.out.println(temp + ", ") ;
    }
}
