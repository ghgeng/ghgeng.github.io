JavaFX Text

提示

在某些情况下，我们需要在应用程序的界面上提供基于文本的信息。JavaFX库为此提供了一个名为JavaFX .scene.text. text的类。这个类提供了各种方法来改变文本的各种属性。我们只需要实例化这个类来在应用程序中实现文本。

# JavaFX文本

属性

Methods Description
setBoundsType(TextBoundsType value) 此属性为对象类型。它决定了计算文本边界的方式。
setFont(Font value) 文本的字体。
setFontSmoothingType(FontSmoothingType value) 为字体定义请求的平滑类型。
setLineSpacing(double spacing) 行与行之间以像素为单位的垂直空间。它是双类型属性。
setStrikeThrough(boolean value) 这是一个布尔类型属性。通过将此属性设置为true，我们可以在文本中插入一行。
setTextAlignment(TextAlignment value) 水平文本对齐。
setTextOrigin(VPos value) 文本坐标系在局部坐标系中的原点。
setText(String value) 它是一个字符串类型的属性。它定义了要显示的文本字符串。
setUnderLine(boolean value) 它是一个布尔类型属性。我们可以通过将这个属性设置为true来在文本中划线。
setWrappingWidth(double value) 它是一个布尔类型属性。我们可以通过将此属性设置为true来为文本加下划线。
setX(double value) X坐标。
setY(double value) y坐标。

# 创建文本节点

为了创建文本节点，需要实例化类javafx.scene.text.Text。使用setter方法setText(string)将字符串设置为文本类对象的文本。按照下面给出的语法实例化Text类。

Text <text_Object> = new Text();   
text.setText(<string-text>);

# 示例1

package code.javafx;

import javafx.application.Application;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.effect.DropShadow;
import javafx.scene.layout.StackPane;
import javafx.scene.paint.Color;
import javafx.scene.shape.QuadCurve;
import javafx.scene.text.Text;
import javafx.stage.Stage;

/**
 * @author :ghgeng
 * @date 2023/9/20 17:39
 */
public class JavaFXText extends Application {

    @Override
    public void start(Stage primaryStage) throws Exception {
        primaryStage.setTitle("JavaFXText");
        Text text = new Text();
        text.setText("Hello");
        StackPane stackPane = new StackPane();
        stackPane.getChildren().add(text);

        Scene scene = new Scene(stackPane, 300, 400);
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

# 字体设置

<text_object>.setFont(Font.font(<String font_family>, <FontWeight>, <FontPosture>, <FontSize>)

  • Family : 它代表字体族。它是字符串类型，应该是系统中存在的适当字体系列。
  • Weight : 这个Font类属性用于字体的粗细。有9个值可以用作字体粗细。FontWeight.BLACK, BOLD, EXTRA_BOLD, EXTRA_LIGHT, LIGHT, MEDIUM, NORMAL, SEMI_BOLD, THIN.(黑色，粗体，超粗体，超浅，浅，中，普通，半粗体，细。)
  • Posture : 这个Font类属性表示字体的姿态。它可以是斜体FontPosture.ITALIC，正常FontPosture.REGULAR
  • Size : 字体大小，这是一个双类型属性。它用于设置字体的大小。

package code.javafx;

import javafx.application.Application;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.effect.DropShadow;
import javafx.scene.layout.StackPane;
import javafx.scene.paint.Color;
import javafx.scene.shape.QuadCurve;
import javafx.scene.text.Font;
import javafx.scene.text.FontPosture;
import javafx.scene.text.FontWeight;
import javafx.scene.text.Text;
import javafx.stage.Stage;

/**
 * @author :ghgeng
 * @date 2023/9/20 17:39
 */
public class JavaFXText extends Application {

    @Override
    public void start(Stage primaryStage) throws Exception {
        primaryStage.setTitle("JavaFXText");
        Text text = new Text();
        text.setX(100);
        text.setY(20);
        text.setText("Hello");
        // 设置字体
        text.setFont(Font.font("Abyssinica SIL", FontWeight.BOLD, FontPosture.REGULAR,20));
        Group group = new Group();
        group.getChildren().add(text);

        Scene scene = new Scene(group, 300, 400);
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

# 描边和颜色

Stroke表示文本边界处的填充。JavaFX允许我们对文本应用描边和颜色。text类提供了一个名为setStroke()的方法，该方法接受Paint类对象作为参数。只需传递将在笔画上绘制的颜色。我们还可以通过将double类型的width值传递给setStrokeWidth()方法来设置笔画的宽度。要设置文本的颜色，javafx.scene.text.Text类提供另一个名为setFill()的方法。我们只需要传递要在文本中填充的颜色。

package code.javafx;

import javafx.application.Application;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.effect.DropShadow;
import javafx.scene.layout.StackPane;
import javafx.scene.paint.Color;
import javafx.scene.shape.QuadCurve;
import javafx.scene.text.Font;
import javafx.scene.text.FontPosture;
import javafx.scene.text.FontWeight;
import javafx.scene.text.Text;
import javafx.stage.Stage;

/**
 * @author :ghgeng
 * @date 2023/9/20 17:39
 */
public class JavaFXText extends Application {

    @Override
    public void start(Stage primaryStage) throws Exception {
        primaryStage.setTitle("JavaFXText");
        Text text = new Text();
        text.setX(100);
        text.setY(20);
        text.setText("Hello");
        // 设置字体
        text.setFont(Font.font("Abyssinica SIL", FontWeight.BOLD, FontPosture.REGULAR,20));
        // 将文本的颜色设置为蓝色
        text.setFill(Color.YELLOW);
        // 设置文字的描边颜色
        text.setStroke(Color.BLACK);
        // 设置描边的宽度为1
        text.setStrokeWidth(1);
        
        Group group = new Group();
        group.getChildren().add(text);
        Scene scene = new Scene(group, 300, 400);
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

# 删除线和下划线

我们可以通过设置javafx.scene.text.Text类的划线和下划线属性来对文本应用装饰。下面给出了这两个方法的语法。

<TextObject>.setStrikeThrough(Boolean value) // true 删除线
<TextObject>.setUnderLine(Boolean value) // true下划线
1
2

package code.javafx;

import javafx.application.Application;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.effect.DropShadow;
import javafx.scene.layout.StackPane;
import javafx.scene.paint.Color;
import javafx.scene.shape.QuadCurve;
import javafx.scene.text.Font;
import javafx.scene.text.FontPosture;
import javafx.scene.text.FontWeight;
import javafx.scene.text.Text;
import javafx.stage.Stage;

/**
 * @author :ghgeng
 * @date 2023/9/20 17:39
 */
public class JavaFXText extends Application {

    @Override
    public void start(Stage primaryStage) throws Exception {
        primaryStage.setTitle("JavaFXText");
        Text text = new Text();
        text.setX(100);
        text.setY(20);
        text.setText("Hello");
        // 设置字体
        text.setFont(Font.font("Abyssinica SIL", FontWeight.BOLD, FontPosture.REGULAR,20));
        // 将文本的颜色设置为蓝色
        text.setFill(Color.YELLOW);
        // 设置文字的描边颜色
        text.setStroke(Color.BLACK);
        // 设置描边的宽度为1
        text.setStrokeWidth(1);
        // 添加下划线
        text.setUnderline(true);


        Text text1 = new Text();
        text1.setX(100);
        text1.setY(60);
        text1.setText("world");
        text1.setFont(Font.font("Abyssinica SIL", FontWeight.BLACK, FontPosture.ITALIC,40));
        // 设置删除线
        text1.setStrikethrough(true);

        Group group = new Group();
        group.getChildren().addAll(text , text1);
        Scene scene = new Scene(group, 300, 400);
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}
