              # 字符串替换

              命令 说明
              :s/before/after/ 替换当前行第一个 before 为 after
              :s/before/after/g 替换当前行所有 before 为 after
              :n,$s/before/after/ 替换第 n 行开始到最后一行中每一行的第一个 before 为 after, n为数字,若 n 为点.，表示从当前行开始到最后一行
              :n,$s/before/after/g 替换第 n 行开始到最后一行中每一行所有 before 为 after, n为数字,若 n 为点.，表示从当前行开始到最后一行
              :%s/before/after/ 等同于:g/before/s//after/替换每一行的第一个 before 为 after
              :%s/before/after/g 等同于 :g/before/s//after/g替换每一行中所有 before 为 after
              :s#before/#after/# 可以使用 # 作为分隔符，此时中间出现的 / 不会作为分隔符,替换当前行第一个 before/ 为 after/
              :%s+/oradata/apras/+/user01/apras1+ 使用加+来替换/ ,/oradata/apras/替换成/user01/apras1/

              # 删除文本中的^M

              提示

              问题描述：对于换行,window下用回车换行(0A0D)来表示，linux下是回车(0A)来表示。这样，将window上的文件拷到unix上用时，总会有个^M.请写个用在unix下的过滤windows文件的换行符(0D)的shell或c程序。

              • 使用命令：cat filename1 | tr -d "^V^M" >newfile
              • 使用命令：sed -e "s/^V^M//" filename > outputNewfileName.txt

              需要注意的是在1、2两种方法中，^V和^M指的是Ctrl+V和Ctrl+M。你必须要手工进行输入，而不是粘贴。

              • 在vi中处理：首先使用vi打开文件，然后按ESC键，接着输入命令：%s/^V^M//。
