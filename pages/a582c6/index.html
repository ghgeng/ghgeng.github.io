Java JFrame

# JFrame

JFrame类是继承java.awt.Frame类的容器类型。JFrame的工作方式类似于主窗口，其中添加了标签、按钮、文本字段等组件来创建GUI。 与Frame不同的是，JFrame可以通过setDefaultCloseOperation(int)方法隐藏或关闭窗口。

# 常用构造函数：

Constructor Description
JFrame() 它构建了一个最初不可见的新框架。
JFrame(GraphicsConfiguration gc) 它在屏幕设备的指定GraphicsConfiguration中创建一个Frame和一个空白标题。
JFrame(String title) 它创建一个新的，最初不可见的框架与指定的标题。
JFrame(String title, GraphicsConfiguration gc) 它创建一个具有指定标题和指定屏幕设备的GraphicsConfiguration的JFrame。

# 常用方法：

Method Description
addImpl(Component comp, Object constraints, int index) 添加指定的子组件。
createRootPane() 由构造函数方法调用以创建默认的rootPane。
frameInit() 由构造函数调用以正确初始化JFrame。
setContentPane(Containe contentPane) 它设置contentPane属性。
setDefaultLookAndFeelDecorated(boolean defaultLookAndFeelDecorated) 提供关于新创建的jframe是否应该具有当前外观提供的窗口装饰(如边框、关闭窗口的小部件、标题……)的提示。
setIconImage(Image image) 它设置要作为此窗口的图标显示的图像。
setJMenuBar(JMenuBar menubar) 它为这个框架设置菜单栏。
setLayeredPane(JLayeredPane layeredPane) 它设置layeredPane属性。
getRootPane() 它返回这个框架的rootPane对象。
getTransferHandler() 它获得transferHandler属性。

# 示例代码1：

package code.swing.learn;

import javax.swing.*;
import java.awt.*;

/**
 * @author :ghgeng
 * @date 2023/7/20 16:09
 */
public class JFrameTest {

    public static void main(String[] args) {
        JFrame jFrame = new JFrame();

        JButton jButton = new JButton("JFrameTest");
        jFrame.add(jButton);

        jFrame.setLayout(new FlowLayout());
        jFrame.setLocationRelativeTo(null);
        jFrame.setVisible(true);
        jFrame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        jFrame.setSize(400, 400);
    }
}
