JavaFX 混合效果

提示

JavaFX Blend
一般来说，混合效果产生的输出是由两个或多个不同输入节点混合而产生的。它获取两个或多个节点的像素，根据应用的混合模式混合它们，并在同一位置产生输出节点。 如果两个图像相互重叠，则混合模式应用于两个图像的重叠区域。

# 属性

方法 描述
setBottomInput(Effect value) 混合操作的底部输入。这是一个对象类型属性。
setMode(BlendMode value) 根据该模式，输入被混合在一起。
setOpacity(double value) 这是double类型的不透明度值。
setTopInput(Effect Value) 混合操作的顶部输入。

# 示例1

package code.javafx.JavaFXEffects;

import javafx.application.Application;
import javafx.scene.Group;
import javafx.scene.Scene;
import javafx.scene.effect.Blend;
import javafx.scene.effect.BlendMode;
import javafx.scene.effect.ColorInput;
import javafx.scene.effect.ImageInput;
import javafx.scene.image.Image;
import javafx.scene.paint.Color;
import javafx.scene.shape.Circle;
import javafx.scene.shape.Rectangle;
import javafx.stage.Stage;

/**
 * @author :ghgeng
 * @date 2023/9/20 17:39
 */
public class JavaFXBlend extends Application {

    @Override
    public void start(Stage primaryStage) throws Exception {
        primaryStage.setTitle("JavaFXBlend");
        Circle circle = new Circle(150,200,120);
        circle.setFill(Color.RED);

        Blend blend = new Blend();
        ColorInput color = new ColorInput(70, 20, 160, 150, Color.LIMEGREEN);
        blend.setTopInput(color);
        blend.setMode(BlendMode.SOFT_LIGHT	);
        circle.setEffect(blend);
        
        Group root = new Group();
        root.getChildren().addAll(circle );
        Scene scene = new Scene(root, 700, 400,Color.WHITE);
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

# 混合模式说明

JavaFX提供了各种混合模式，可以用来修改混合效果。

混合模式 描述 输出
ADD 顶部输入的颜色组件被添加到底部输入的颜色组件中。
BLUE 只有底部输入的蓝色分量被顶部输入的蓝色分量所取代。
COLOR_BURN 底部的输入颜色被反转并除以顶部的输入颜色分量。结果再次反转以获得输出颜色。
COLOR_DODGE(颜色减淡) 顶部颜色组件被倒置并划分底部颜色组件以产生输出颜色。
DARKEN(变暗) 选择两个输入分量颜色中较深的颜色来产生结果颜色。
DIFFERENCE(差异) 两种输入颜色中的较深的颜色从较浅的颜色中减去以产生结果颜色。
EXCLUSION(排除) 将两个输入颜色分量相乘和加倍，然后从底部颜色分量的总和中减去，以产生所需的颜色。
GREEN(绿色) 底部输入的绿色分量被顶部输入的绿色分量所取代。
HARD_LIGHT(硬光) 根据底部颜色的不同，输入颜色组件要么成倍增加，要么进行筛选。
LIGHTEN(发亮) 两色分量中较浅的颜色作为输出产生。
MULTIPLY(正片叠底) 这两种颜色成分相乘产生输出颜色。
OVERLAY(覆盖) 根据底部颜色的不同，输入颜色组件要么被筛选，要么被相乘。
RED(红色) 底部输入的红色分量被顶部输入的红色分量所取代。
SCREEN 这两种颜色的组成部分被颠倒，相乘，再颠倒，以产生所需的结果。
SOFT_LIGHT(柔光) 输入颜色组件变亮或变暗。
SRC_ATOP 顶部输入的部分位于底部输入的部分被混合。
SRC_OVER 顶部输入与底部输入混合。
